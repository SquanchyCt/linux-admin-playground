### Среда
Docker Engine\n
Ubuntu Server

### Начало

Запускаем команду:
docker run -dit --name UbuntuServer ubuntu:24.04 - запуск контейнера с образом ubuntu:24.04\n
docker exec -it UbuntuServer bash - открытие контейнера в интерактивном терминале c bash консолью

### Создание файловой системы

1. Прежде чем начать создавать каталоги, посмотрим где мы находимся:\n
    ls - команда показывающая какие каталоги доступны для открытия и в какой находимся мы
2. Создание каталогов:\n
    cd /opt - заходим в каталог /opt\n
    mkdir devops-app - создаем каталог devops-app\n
    cd .. - выходим на директорию ниже, отсюда будем создавать все остальные директории прописывая полный путь\n
    mkdir /logs\n
    cp -r /logs /backups - копируем каталог /logs(с флагом рекурсии) и создаем на его основе новый каталог с именем /backups\n
    cp -r /logs /scripts
3. Поиск файлов .log и текста по всем файлам\n
    Для примера попробуем найти все файлы ".log" больше 10МВ:\n
    find . -name ".log" -size +10M - у меня ничего не нашлось, но команда сработала\n
    Попробуем найти слово "error" в файлах директории /var/log:\n
    grep -r "error" /var/log -выведет все совпадения в файлах директории /var/log\n
